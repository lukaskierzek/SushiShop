## Technologies Used
- .NET 8
- Entity Framework Core 8
- Vite 4
- React 18
- Visual Studio 2022
- MS SQL 2019

## Features
- You can browse sushi by categories.
- You can browse sushi by ingredients.
- Each sushi has a subpage with a detailed description.
